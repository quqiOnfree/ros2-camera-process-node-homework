## 任务目标

- **配置 ROS2 开发环境**：在 Ubuntu 系统中安装并配置 ROS2。
- **安装并运行 USB 相机驱动**：使用 `usb_cam` 功能包驱动 USB 摄像头或电脑内置相机节点，并发布图像话题。
- **编写 detect 功能包**：实现一个数字识别节点，订阅相机图像，对你制作的数字卡片（手写/打印均可）进行分类。
- **发布识别结果话题**：将分类结果发布为一个 ROS2 话题，供其他节点订阅使用。
- **提交任务源代码、演示图片和视频以及使用方法思路的**README：自行创建仓库并上传到 Github，将 Github 链接发送至 1416886302@qq.com

## **核心实现**

### 安装与使用 usb_cam

### `usb_cam`: https://github.com/ros-drivers/usb_cam

将其克隆到工作空间 `src` 下，并编译

### detect 功能包实现要求

detect 包需实现一个节点，完成：

- 订阅相机图像
- 对图像中的数字卡片进行识别（0-9，或你自定义的其他类别也可，但必须在 README 说明）
- 发布识别结果话题

#### 话题与消息类型

- 订阅：`/image_raw`（或你实际相机话题）
- 发布：`/digit_class`（发布数字类别）

你也可以同时发布置信度，例如：

- `/digit_class`：`Int32`
- `/digit_score`：`Float32`

### Launch 

你需要提供一个 `launch/detect.launch.py`，能一键启动：

- `usb_cam` 相机节点
- `detect_node` 节点

#### 项目构建与运行

运行后，将实时的相机图像和实时的话题数据录制视频
